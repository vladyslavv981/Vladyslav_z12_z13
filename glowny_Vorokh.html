Definicja layoutu (interfejs strony internetowej):

Layout to element konstrukcji graficznej, w którym ustala się wygląd (kolorystykę, elementy dekoracyjne, krój czcionki) i rozmieszczenie elementów (treści) na stronie. W fazie projektowej może mieć postać pliku graficznego prezentującego zakładany styl.

Popularne technologie layoutu dla stron WWW:

  1. CSS Grid - Tworzenie złożonych, responsywnych układów w dwóch wymiarach (wiersze i kolumny).
  2. Flexbox - Układ w jednym wymiarze (poziomo lub pionowo), idealny do wyrównywania elementów.
  3. Frameworki CSS - Gotowe systemy, np. Bootstrap, Foundation, zapewniające siatki i responsywność.
  4. Responsive Web Design (RWD) - Dostosowanie układu do różnych urządzeń za pomocą media queries.
  5. Preprocesory CSS - Sass czy LESS, upraszczające tworzenie złożonych stylów i układów.
